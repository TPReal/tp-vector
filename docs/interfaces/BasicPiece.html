Interface BasicPiece

The most basic part of a project defining a geometry. It is Layerable, it can contain Defs, and it can produce SVGElement's. Additionally, its bounding box might be defined separately.

selectLayers

  • selectLayers(...layers: OrArrayRest<any>): BasicPiece

  • Returns a subset of this object that is assigned to any of the specified layers. If NO_LAYER is specified, includes any parts not assigned to any layer.

    If this object is assigned directly (using setLayer) to one of the specified layers, then this object is returned with all of its parts included. If this object is assigned directly to some different layer then empty object is returned. If this object is not assigned directly to any layer (or assigned to NO_LAYER), selection recurses into its parts.
