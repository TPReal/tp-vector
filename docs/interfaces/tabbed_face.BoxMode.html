Interface BoxMode

Box mode configuration. The box mode allows to easily create e.g. the base (bottom) face of a box in the shape of a right prism. In this mode, on each turn the box correction is applied.

Note: In box mode it might be difficult to track the current position of the turtle, as the corrections are applied. Behaviour can also be surprising at times, e.g. .right(0) moves the Turtle forward (by the tab width). That's why it is disabled by default.
