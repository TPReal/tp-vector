getRunsInNaturalOrder

  • getRunsInNaturalOrder(): PartialRunsSelector[]

  • If preserveRunsOrder was specified, the natural order is the same as the specified order. Otherwise returns all the runs defined in this Sheet in their natural order. The order is:

    • prints on the back side,
    • cuts on the back side (for scoring, as the main cut needs to be done on the front side),
    • the reversing frame - if there were any runs on the back,
    • prints on the front,
    • cuts on the front.

    Returns PartialRunsSelector[]
