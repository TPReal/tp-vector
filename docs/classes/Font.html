Class Font

An AttributesDefTool is a tool that can be used on a piece to change its properties by setting particular attributes and attaching a Defs to it. It is typically used for things like clip path, mask or font.

Static googleFonts

  • googleFonts(name: string, fontAttributes?: FontAttributes): Promise<Font>

  • Creates a font from Google Fonts. The font is fetched and encoded as data URI.

    If the font is not available on Google Fonts with the specified attributes, the fetch will fail. It is still possible to use faux attributes, with:

    Font.googleFonts(name).setFontAttributes(fontAttributes)
