Class SimpleLazyPieceAbstract

A superclass for custom classes that wants to extend Piece in a lazy way, i.e. so that the actual geometry is only created on demand. This is useful for builder-like classes if building the SVG geometry after each build step would be expensive.

For a usage example, see the Path class.
