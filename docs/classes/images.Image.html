  • selectLayers(...layers: OrArrayRest<any>): Piece

    Returns a subset of this object that is assigned to any of the specified layers. If NO_LAYER is specified, includes any parts not assigned to any layer.

    If this object is assigned directly (using setLayer) to one of the specified layers, then this object is returned with all of its parts included.
